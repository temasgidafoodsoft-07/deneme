# Temaş Foodsoft Eğitim - GitHub Pages

Bu klasör doğrudan GitHub Pages üzerinde çalışacak şekilde hazırlanmıştır.
Vercel ve Render gerekmez.

## Yayınlama

1. GitHub'da yeni ve boş bir depo oluşturun.
2. Bu paketin içindeki dosyaları deponun ana dizinine yükleyin.
3. `Settings > Pages` bölümünü açın.
4. `Source` alanında `Deploy from a branch` seçin.
5. Branch olarak `main`, klasör olarak `/(root)` seçip `Save` düğmesine basın.

Bir iki dakika sonra GitHub, sitenin bağlantısını aynı ekranda gösterir.

Ana uygulama:

`https://KULLANICI-ADINIZ.github.io/DEPO-ADINIZ/`

Yönetim sayfası:

`https://KULLANICI-ADINIZ.github.io/DEPO-ADINIZ/admin.html`

## Önemli

Eğitim sayfaları ve görseller yalnızca GitHub Pages ile açılır. Kullanıcıların
farklı cihazlarda ortak yönetilmesi istenirse yönetim ekranındaki Supabase
bağlantısı ayrıca kullanılmalıdır. Supabase olmadan yerel kullanıcılar yalnızca
oluşturuldukları tarayıcıda saklanır.
